# form_builder
**A dynamic Form Builder Web Application built with PHP, MySQL, JavaScript, HTML & CSS.
This system allows users to create custom forms using drag & drop, generate shareable form links,
collect responses, send form data to email, and manage all submissions from an admin dashboard**
<br><br><br>
**Login → Dashboard → Create Form (Drag & Drop Fields) → Save Form** <br>
                         ↓  
   **Generate Form Link**  
                         ↓  
   **User Opens Link → Fills Form → Submit**  
                         ↓  
 **Thank You Message → Data Sent to Email**  
                         ↓  
  **Admin Can View All Responses in Panel**

  <br><br>
  **Dashboard** <br><br>
  **A centralized dashboard for managing form creation, drag-and-drop fields, shareable links, and viewing user submissions.**  <br><br><br>

**🧲 Drag & Drop Form Builder** <br><br>
**The drag and drop module allows users to create custom forms visually without writing code. Users can drag different field types (such as text input, email, textarea, etc.) 
and drop them into the form area. Each field can have a custom label and validation. This makes form creation easy, fast, and user-friendly.** <br><br><br>

**🔗 Open Link (Public Form Access)** <br><br>
**The Data Access layer handles all communication between the application and the MySQL database. It stores form structures, generated links, and user responses.
This ensures that form data is saved securely and can be retrieved later for display in the admin panel.** <br><br><br>
